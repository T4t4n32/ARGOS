# Documentación de ARGOS

Este directorio alberga la documentación técnica y académica del sistema ARGOS. Cada subcarpeta contiene un grupo de documentos organizado por temática.

## Estructura

- **architecture/**: descripciones de la arquitectura del sistema, diagramas de bloques y flujos operativos.
- **safety/**: protocolos de seguridad, advertencias y umbrales operativos.
- **templates/**: plantillas para cronogramas, listas de materiales (BOM), planes de prueba y bitácoras.
- **thesis/**: capítulos del proyecto académico y su desarrollo en formato Markdown.
- **references/**: referencias bibliográficas en formatos APA/IEEE.
- **identity/**: manual de uso de logotipos y activos visuales del proyecto.

Cada sección debe contener un README específico que describa su propósito y relación con las demás partes del proyecto.
